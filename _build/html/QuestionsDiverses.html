Questions diverses¶

Les hotspots : Hotspot dual band = on peut configurer deux fréquences différentes pour RX et TX, le hotspot passe pour relais Hotspot single band = Mode dmo. Utiliser une seule fréquence RX/TX sinon pas de cnx possible au master. Le master vérifie la config. Pas necessaire d’avoir un ID à 6 chiffres pour utiliser hotspot Utiliser un SSID par hotspot, ça permet de différencier les configurations dans selfcare. Dès qu’on a un ssid, le hotspot arrive direct dans le selfcare. Si des ssid ont été déclarés et ne sont pas utilisés, il sont nettoyés tous les 3 mois.
